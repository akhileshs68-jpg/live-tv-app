# Favorite Channels - User Guide

## How to Use Favorites

### Saving a Channel

1. **Browse home page** - See all available live TV channels
2. **Hover over any channel** - A heart icon appears in top-right
3. **Click the heart** - Channel is instantly saved
4. **Heart fills up** - Visual confirmation it's saved

### Viewing Favorites

#### Option 1: Home Page
- Scroll to top of home page
- See "Favorite Channels" section
- All your saved channels appear here
- Click play button to watch immediately

#### Option 2: Favorites Page
- Click **"Favorites"** in navigation menu (bottom on mobile)
- See all saved channels in organized grid
- Click any channel to play it
- More details and management options

### Managing Favorites

#### Remove a Channel
1. Click the heart icon again on the channel, OR
2. Go to /favorites page
3. Hover over channel and click **X** button
4. Channel removed from favorites

#### Remove All Favorites
1. Go to /favorites page
2. Click **"Clear All"** button (top right)
3. Confirm in dialog
4. All favorites removed

#### Play from Favorites
1. Find channel in Favorites section
2. Click **play button** (big circle icon)
3. Video player opens instantly
4. No need to search!

## Troubleshooting

### Favorites not saving?
- Check browser localStorage is enabled
- Try refreshing the page
- Ensure you're not in private/incognito mode

### Favorites disappeared?
- Check if you cleared browser data/cookies
- Try adding again
- Your favorites are stored locally on this device only

### Can't access /favorites page?
- Make sure you're logged in
- Click "Favorites" from navigation menu instead
- Try refreshing the page
